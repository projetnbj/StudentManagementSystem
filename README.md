# StudentManagementSystem
 The system is a java console application that manages students' results with persistent storage

 # What it does
 - Register new student
 - Add new subjects
 - Calculate the average result for each student

## Technologies used
- Java 22
- SQLite (database)
- JDBC (database connection)
- Maven (project management
  
## How to run
1. Clone the repository
2. Open the project in NetBeans
3. Add the SQLite dependency to pom.xml
4. Run the project

## What I learned
- Object oriented programming in Java
- ArrayList
- Database integration with JDBC
- SQL injection prevention with PreparedStatement
- Data persistence with SQLite
